## 📑프로젝트 설명
* 컬쳐모아는 최신 행사 정보를 받아보고, 공연에 관심있는 사람들과 소통할 수 있는 사이트입니다.
행사 일정, 장소, 티켓 오픈 날짜, 주차장 보유 여부 등의 정보를 제공합니다.
또한 쪽지, 게시판, 오픈채팅 등 다양한 커뮤니티를 운영합니다.
